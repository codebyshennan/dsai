# Decision Trees

**After these lessons, you can build and interpret decision trees for classification and regression, prune for generalisation, and recognise when trees outperform simpler models.**

## Overview

Decision trees split data recursively on feature thresholds, producing an interpretable flowchart-like structure. Prerequisites: [5.1 Intro to ML](../../5.1-intro-to-ml/README.md).
