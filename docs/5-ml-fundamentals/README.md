# Module 5: Machine Learning Fundamentals

**After this module, you can build, evaluate, and tune supervised and unsupervised ML models using scikit-learn, and explain the reasoning behind algorithm and metric choices.**

## Overview

Covers the full ML workflow from algorithm intuition through model evaluation best practices. Prerequisites: Modules 1–4.

## Lesson Path

| Order | Submodule | Focus |
|-------|-----------|-------|
| 5.1 | [Intro to ML](5.1-intro-to-ml/README.md) | Workflow, features, bias–variance |
| 5.2 | [Supervised Learning 1](5.2-supervised-learning-1/README.md) | Naive Bayes, kNN, SVM, Decision Trees |
| 5.3 | [Supervised Learning 2](5.3-supervised-learning-2/README.md) | Random Forest, Gradient Boosting, Neural Networks |
| 5.4 | [Unsupervised Learning](5.4-unsupervised-learning/README.md) | Clustering, PCA, dimensionality reduction |
| 5.5 | [Model Evaluation](5.5-model-eval/README.md) | Metrics, cross-validation, hyperparameter tuning |

After Module 5, continue to [Module 6: Capstone](../6-capstone/README.md).
