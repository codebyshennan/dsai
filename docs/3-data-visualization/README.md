## Overview

By the end of this module, you should be able to choose chart types for a goal, build clear static and interactive visuals in Python, use BI-style tools for dashboards, and present findings as a short data story.

> **Note:** Submodules mix **code-first** lessons (Matplotlib, Seaborn, Plotly) and **UI-first** lessons (Tableau, Looker Studio, Power BI). Each lesson states which style it uses.

## Prerequisites

- Comfortable with Python, **pandas**, and basic **EDA** from Modules 1 and 2.
- Optional: Jupyter or VS Code for running notebook-style examples.

## Code blocks (maintainer pattern)

Lesson pages in this module pair substantive fenced blocks with a short **title**, **Purpose:** (what the learner should get from the snippet), and optional **Walkthrough:** (lines/APIs/steps that matter). Diagram-only `yaml` or ASCII blocks are labeled as non-code. Notebook stdout (for example Matplotlib return values) is called out in prose so it is not mistaken for code to copy. When adding new snippets, follow the same shape so bulk or scripted edits stay consistent.
